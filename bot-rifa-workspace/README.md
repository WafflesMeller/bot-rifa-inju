# Bot Rifa - Área de trabajo

Proyecto monorepo mínimo con backend (Node.js + Express) y frontend (React + Vite).

Pasos rápidos:

- Copiar `.env.example` a `.env` en `backend` y rellenar las variables.
- Instalar dependencias en `backend` y `frontend` con `npm install`.
- Ejecutar backend: `npm run dev` desde `backend`.
- Ejecutar frontend: `npm run dev` desde `frontend`.

Backend: `bot-rifa-workspace/backend`
Frontend: `bot-rifa-workspace/frontend`
